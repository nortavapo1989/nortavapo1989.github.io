    This message appears if you do not have the latest releases of the game, i.e. versions 2.5.xx
    If the original game already provides you with problems and crashes, it makes no sense to try and mod it.
    Another source of problems are windows versions prior windows 7. They will not work.
    Also check for the following
    - your game folders contain a file called engine.lua, open it with an editor to see its content. There is a name of a folder in your user/documents
    - check your user/documents/ for folders for Baldurs Gate 1 and 2. Make sure these folders have a baldur.lua file
    Check with your task manager that no copy of the game is running in the background while you try to mod the game.
